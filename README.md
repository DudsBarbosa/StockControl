# Stock Control

## Case
- Controle de estoque de peças onde o sistema alerta o usuário sobre saldo de estoque indisponível ou peça inexistente.
- Não permite consumir mais do que o disponível.
- Retorna o quanto de peças foi consumido no dia e seu respectivo custo a partir do preço médio de custo colocado no 
produto.
- Inclui o part number da peça.

## Technologies, architecures and patterns

- .NET 8
- MySQL
- Dapper
- Dependency Injection
- TDD
- Clean Code
- CQRS e MediatR
